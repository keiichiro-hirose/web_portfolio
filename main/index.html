自己紹介

自分の手で何かを作ることが好きなソフトウェアエンジニアです。

新卒でSIerに入り、転職してベンチャーに行きました。その後ブランクを経て、もういちどwebエンジニアとして再出発したいと考えております。

ベンチャーでは自社製品開発を行っておりました。GitHubを使ったアジャイル開発で、開発環境としてはメンバーズエッジ様と近いものと思われます。チャットツールにも慣れておりますので、業務にはすんなり入っていけると思います。

目の前の課題を解決するためなら、積極的に新しい技術を学んでゆく姿勢を持っています。結果として、複数の言語に親しむことになりました。メンバーズエッジ様では、ScalaあるいはJavaScriptでお役に立てるかと思います。また、プロジェクトに必要であればPHPもやっていこうと考えています。

今まで長く同じ所で働いた経験はありませんが、今回は長く（できれば定年まで）安心して働ける会社を探しています。ワークライフバランスを重視しつつ、東京から離れてリモートで働ける職を探しており、メンバーズエッジ様はその点で理想的な職場と考えています。

以下に今回のためのデモを用意しましたので、ご覧いただけると幸いです。

WORKS

TOKYO BALL

ブラウザで動くオンラインCo-opシューターです。複数人で協力してハイスコアを目指します。

今回求職するにあたって、なにか見せられるものがあるといいなと思い、作りました。node.jsとSocket.IOで動いています。軽く触っていただけると幸いです。

自動化ツール

WindowsPCの組み立て時に使用する、設定/負荷テストツールです。cmd, PoworShell, C#を組み合わせて作っています。

アルバイト先ではこれを一台ずつ手作業でポチポチやっていたので、我慢できずに作りました。C#は触ったことがなかったので、良い勉強になりました。ご好評につき、現在も使っていただいています。

Scala 業務でApache Sparkの分散処理をやっていました。広告配信のプロジェクトがあるとのことで、お役に立てると思います。
JavaScript ブラウザオンラインゲームを作るために勉強しました。node.jsで動いています。(上記参照)

ほか AWS, Git, Ubuntu, java, HTML, CSS, SQLなど、浅く広く。
